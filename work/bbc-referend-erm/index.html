BBC Referend-erm?

This project was a mobile first, responsive website for the upcoming Scottish Referendum. The whole point of this project was to build a platform that is an informative, interactive space for 16-19 year olds where no question is too big, small or 'stupid' to ask.

I worked on this project during my time contracting at Realise Ltd. I was responsible for turning the original prototype into a fully-functioning website, build on-top of WordPress for easy update with the latest news and questions regarding the Scottish Referendum.

Credit where credit is due

The concept, design and build are by Realise, and the project was conceived through the BBC Connected Studio Referendum programme.

Chris Rush at Realise Ltd was the man behind this delightful and engaging user interface, and Steve Robinson (Senior Developer) was responsible for the prototype of this project as well as supporting me throughout the build.

Imagery on this page was created by Chris Rush, whose write-up of this project can be seen here.
